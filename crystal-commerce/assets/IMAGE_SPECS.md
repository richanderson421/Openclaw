# Image Specs (Crystal Content Blocks)

Use these as baseline specs to keep homepage visuals clean and consistent:

- Hero banner: 1600x900 (16:9)
- Promo tile: 1080x1080 (1:1)
- Event strip image: 1200x675 (16:9)
- Thumbnail/icon: 512x512

Guidelines:
- Keep key text centered/safe for mobile crop.
- Avoid tiny text baked into images.
- Use high contrast for legibility.
