# Forge Replatform Branding Guide (Draft v1)

## Brand direction
**Positioning:** Modern community hub for tabletop players — welcoming, energetic, and trustworthy.

**Tone:**
- Confident, clear, no fluff
- Friendly to new players
- Community-forward, not gatekeep-y

## Visual system

### Color palette
- **Forge Ember (primary):** `#F97316`
- **Forge Ember Dark (hover/accent):** `#EA580C`
- **Forge Charcoal (background):** `#111827`
- **Forge Slate (secondary bg):** `#1F2937`
- **Forge Mist (light bg):** `#F8FAFC`
- **Forge Ink (body text):** `#0F172A`
- **Forge Steel (muted text):** `#64748B`
- **Forge Gold (highlight):** `#F59E0B`

### Typography
- Sans first (clean + modern)
- Headings: bold, compact line-height
- Body: medium contrast, comfortable spacing

### UI shape and depth
- Radius: 14–20px
- Shadows: soft but noticeable (`shadow-lg` for hero cards, `shadow-sm` for list cards)
- Borders: subtle, cool-gray

## UX principles
1. **Mobile-first always** (most customers discover events on phones)
2. **One-screen clarity** for top actions
3. **Event-forward navigation**
4. **New-player confidence** (clear “start here” path)

## Homepage structure (recommended)
1. Hero: value prop + 2 CTAs
2. Quick action rail
3. Weekly highlights card
4. Category quick links
5. Community + trust footer

## CTA standards
- Primary: solid ember button
- Secondary: outline/slate button
- Use action verbs: “See Events”, “Shop New Arrivals”, “Start Here”

## Accessibility baseline
- Minimum text contrast 4.5:1
- Tap targets >= 44px height
- No critical info conveyed by color alone
